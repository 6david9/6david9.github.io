在终端中使用 swift

1、有一种方法是直接把 swift 的路径加入到 PATH 环境变量中 2、使用 xcrun 命令

修改 PATH

将以下路径加入到环境变量中：

/Applications/Xcode6-Beta.app/Contents/Developer/Toolchains/XcodeDefault.xctoolchain/usr/bin

在命令行执行打开 .bash_profile 文件：

$:vim ~/.bash_profile

在打开的文件中中添加 :

PATH=$PATH:/Applications/Xcode6-Beta.app/Contents/Developer/Toolchains/XcodeDefault.xctoolchain/usr/bin

保存退出就行了。

然后重启下终端或 source 下刚才编辑的那个文件。

使用 xcrun 命令

如果你不想改 PATH，或者只是偶尔使下 swift 命令，那么可以用 xcrun 命令。

$:xcrun swift
