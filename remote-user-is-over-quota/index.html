remote: User Is Over Quota

今天在镜像 AOSP 时，出现了了 remote: User Is Over Quota 这个提示，以为是网络错误，google 了下原来是我下载的速度太快了超过了服务器匿名连接的限额，登录下 google账号配置下就行了。

访问以下网址按提示操作：

https://android.googlesource.com/new-password

具体参考：
https://groups.google.com/forum/#!topic/android-building/_975cuUxVTw
