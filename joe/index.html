大文件编辑器 joe（Joe's Own Editor）

推荐一个 Unix/Linux 下的大文件编辑器 joe，在我的 Mac 上打开一个 2.9G 的文件，占用内存60M+，打开时间半分钟左右，支持 emacs 的语法，可以通过正则表达式查找，对于想要交互式的编辑文件的用户推荐使用，对于只喜欢用 grep，sed，awk 系的用户可以忽略。Mac 下安装方法：

brew install joe

其他的 Unix 或 Linux 用户可以使用对应平台的包管理器安装，或者从源码编译。

关于如何使用，只要记住一个快捷键就行了：

ctrl+k+h

emacs 兼容的快捷键！
