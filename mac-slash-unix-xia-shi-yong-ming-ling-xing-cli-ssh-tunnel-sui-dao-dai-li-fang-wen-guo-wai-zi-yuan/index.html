Mac/Unix 下使用命令行(CLI) SSH Tunnel(隧道) 代理访问国外资源

在 Mac 的终端(terminal)下使用 SSH 隧道访问国外的网站需要安装 proxychains-ng。主要是用于 install 一些安装包时需要，当然了要是使用 wget 一些访问受限的资源也可以。

使用方法：

proxychains4 + 命令
proxychains4 gem install xxx

建立 ssh 隧道也很简单，建立SSH隧道（SSH端口转发）
当然大 Mac 下还有更简单的方法 GoAgentX。当然了有了 GoAgentX 谁还用 CLI 下的代理。。。。。咳咳。。。

安装 proxychains 在 Mac 下可以使用 HomeBrew
HomeBrew 的安装方法：

ruby -e "$(curl -fsSL https://raw.github.com/mxcl/homebrew/go)"

安装好 HomeBrew 后就可以安装 proxychains 了:

brew install proxychains-ng

安装完 proxychains 后需要配置下：

vim /usr/local/Cellar/proxychains-ng/4.6/etc/proxychains.conf

打开后在文件最底部改为：

[ProxyList]
socks5     127.0.0.1 7070    # 127.0.0.1 为本地地址，7070 为开启隧道的端口

并把下面的这行解注释，DNS 解析也使用远程服务器的：

# Proxy DNS requests - no leak for DNS data
proxy_dns
