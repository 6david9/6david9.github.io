Cocoapods 的 Specs 的国内镜像

Cocoapods 的 Specs 项目在 github 上，长久以来访问速度很慢，所以就做了个国内的镜像，每半小时同步一次。 替换方法:

pod repo remove master
pod repo add master https://git.oschina.net/6david9/Specs.git
pod repo list

如果想替换 Ruby gems 的安装源，可以参考 http://ruby.taobao.org/
