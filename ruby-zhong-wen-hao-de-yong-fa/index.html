Ruby 中问号后跟一个字符的用法

今天在读 Sinatra 的源码时看到了这样的用法：

type = ".#{type}" unless type.to_s[0] == ?.

最开始以为是最后那个 ?. 是个正则表达式的特殊写法，后来谷歌了下 ? 后跟一个 ASCII 字符会返回这个 ASCII 字符。

等价的表达式：
?. == '.'
?a == 'a'
?A == 'A'

类似的有 0X 开头的会返回数字的 16 进制形式， 0 开头的会返回 8 进制形式。

Numbers

123
1_234
123.45
1.2e-3
0xffff   # hex
0b01011  # binary
0377     # octal
?a       # ASCII character (1.8 only -- 1.9 returns a string "a")
?\C-a    # Control-a
?\M-a    # Meta-a
?\M-\C-a # Meta-Control-a
